# data-harga
Data harga-harga berbagai layanan atau komoditas yang dikumpulkan user GitHub bennyesp.

## Penggunaan
Repo ini menyediakan berbagai data harga untuk blog BP Blog ([bennypadang.com](https://www.bennypadang.com)) dan setiap subdomainnya.

## Daftar Harga
### BBM
Daftar harga yang digunakan BP Blog dalam perhitungan biaya perjalanan, misalnya pada [Penghitung Kebutuhan BBM dalam perjalanan dan Biayanya](https://myapp.bennypadang.com/bill/kebutuhan-bbm) ini

### Penyesuaian
Harga BBM non-subsidi mengalami penyesuaian secara rutin pada tanggal 1 setiap bulan, sehingga update akan dilakukan pada minggu pertama dalam setiap bulan.

Harga BBM Subsidi akan disesuaikan berdasarkan prosedur tertentu yang ditetapkan pemerintah. Jika suatu saat akan ada penyesuaian harga BBM subsidi pada tanggal selain 1, maka data ini akan segera diupdate.
